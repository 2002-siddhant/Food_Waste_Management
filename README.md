# Food Waste Management

A college-level full-stack web application for managing surplus food donation workflows.

## What this project does

This app connects four user roles:
- Donor: posts surplus food donations
- NGO: views and accepts donations
- Delivery Partner: receives assigned deliveries
- Admin: manages users, donations, assignments, and reports

## Core features
- User registration and login
- Role-based access control
- Donation creation and tracking
- NGO acceptance workflow
- Admin delivery assignment workflow
- Delivery partner status updates
- Donor lifecycle views for active, past, and rejected donations

## Tech stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- Authentication: JWT

## Project structure
- Frontend: React UI and routing
- Backend: REST APIs, auth, role-based logic, MongoDB models

## How to run locally
1. Start MongoDB
2. Open the backend folder and install dependencies
3. Start the backend server
4. Open the frontend folder and install dependencies
5. Start the frontend development server

## Interview-friendly explanation
You can explain this project as:
> A full-stack food donation platform where donors post food, NGOs accept it, admins assign delivery partners, and delivery partners complete the handoff. The project demonstrates authentication, role-based access, CRUD operations, state management, and a multi-step workflow.
